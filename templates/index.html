    Datatypes

Registers contain facts the law says government must record, which meets a user need and may be cited as the official source of truth. Example registers include: address, country, postcode, post-town, and school.

Entries in a register consists of a set of fields. Example fields include: name, text, longitude, latitude, start-date, and end-date,

Here you can see demonstration applications built using the register API.

This alpha is being developed by a small team at the Government Digital Service as open source software. We welcome contributions and feedback.
